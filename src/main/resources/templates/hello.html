A Little Detail

Most of the social media links are already added so I don't have to write much. Feel free to browse.

This website is just experimental, as I wanted to host something on Google cloud. Google cloud tutorial has been a great help to quickly learn and host this one-page website. This is running on Google app engine flexible platform on tomcat server. I had to use the flexible platform option as this application is spring boot based which generated a jar with embedded tomcat.I don't want to create war file just to utilise standard platform

In future, I might move it to Google cloud standard platform to get high responsiveness. If this page took time to load means its still not yet migrated and compute box took time to wake up. Another subsequent request will return quickly

Also check out semantic-ui , I am using CSS from there.
